Convention Tycoon

Unity, C# (2018 - present)




  • This is a tile-based isometric building and management game.
  • An in-game tutorial explains the game’s mechanics and helps the player become oriented with the UI.
  • The player can place or destroy floors, walls, and facilities on a grid. The game provides visual feedback indicating valid or invalid placement.
  • Characters use A* pathfinding to reach destinations based on their personal goals, which are determined using weighted probabilities and “character type.” “Staff” build and maintain facilities, while “guests” visit attractions and spend money.
  • Motion-tween style character animations were created directly in Unity.
  • Outfits are procedurally generated, and the player may customize their own character.
  • To help ensure a smooth workflow when adding new content, furniture and clothing items are defined in XML. The game uses those XML files to populate in-game lists of items, and to determine the correct sprites to use during gameplay.
  • The game can be saved and loaded using XML. Safeguards are in place to help increase compatibility with save files from old versions of the game.






Mahjong Solitaire: Junk Journal Edition

Unity, C# (2019)




  • This is a casual matching game, with logic created from the ground up to provide the Mahjong Solitaire experience that seasoned players will expect.
  • The player chooses between several different layouts, which are loaded from plain text files. Layouts have an x, y, and z component. Based on the chosen layout, the game generates a random winnable level.
  • The player can choose from several tile sets, or allow the game to choose one at random. A developer can quickly add a new set with any number of unique tile designs (be it 2 or 200), and it will still work with the level generation.
  • The game features a hint button, a message if the player can no longer possibly clear the rest of the tiles, and visual and audio feedback when tiles are selected and matched.
